⚠ DADOS SENSÍVEIS E SIGILO PROCESSUAL
Esta ferramenta processa textos de processos judiciais que podem conter dados pessoais sensíveis, segredo de justiça, informações de menores, dados médicos e financeiros protegidos pela LGPD (Lei 13.709/2018) e pelo Código de Ética da OAB. 🔴 ATENÇÃO CRÍTICA: Ao usar o Scanner Forense, o texto do processo é enviado para a API do Claude (Anthropic) para análise. Isso significa que o conteúdo trafega pela infraestrutura de uma empresa americana. NÃO use esta ferramenta com processos sob segredo de justiça absoluto ou que contenham dados de vítimas de crimes sexuais, menores de idade ou informações sigilosas de Estado. Para esses casos, use apenas a geração de prompt (que não envia dados para API externa).
